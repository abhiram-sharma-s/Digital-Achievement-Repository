Digital Achievement Repository (DAR) is a web-based application designed to store, manage, and verify academic achievements of students and faculty within an educational institution.
The system enables users to upload, view, update, and delete their certificates and badges, while administrators can monitor, verify, and manage all uploaded documents.

It aims to eliminate paperwork, prevent data loss, and create a secure digital space for institutional achievements.
